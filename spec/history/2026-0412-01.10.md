# バージョン 2026-0412-01.10

## 概要
徹底点検と「人間系ミス」へのガードレール強化。パースの堅牢化と説明文同期のサポート。

## 変更・追加内容
### 堅牢化プログラミング (Core)
- **パースロジックの刷新**: Markdown 形式（`- [ ] **Title** (Range) (ID: id)`）のパースにおいて、タイトル内にカッコが含まれていても正確に ID を抽出できるよう、後方一致ベースのロジックに改良。
- **説明文 (Description) のインポート対応**: Markdown のリストにぶら下がるインデント行を、カレンダーの「説明文」として読み取り、同期する機能を追加。
- **一括バリデーション**:
  - 時刻の逆転（開始 > 終了）の検知。
  - 同一ファイル内での ID 重複に対する警告と保護。

### 安全性の強化
- **プライマリ保護**: `cal delete primary` を制限し、メインカレンダーの削除を防止。
- **パース失敗の通知**: 形式不正な行をサイレントに無視せず、行番号とともにユーザーへ警告。

### テスト
- **TC-08 〜 TC-13**: 特殊記号タイトル、ID重複、時刻逆転、削除保護などの過酷なテストケースを追加し、すべて合格（または正しくバリデーションで停止）することを確認。
