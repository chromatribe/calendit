# 変更履歴 `2026-0424-01.02`

## EventKit 常駐ブリッジ（MVP）

- **`native/eventkit-bridge`**: SwiftPM 実行ファイル `eventkit-bridge serve`。Unix ソケットで JSON 1 行リクエスト（`v` / `op` / `token` / `body`）。起動時に `bridge.token`（`0600`）を生成。
- **`src/core/eventkitHelper.ts`**: `CALENDIT_EVENTKIT_BRIDGE=1` または `unix:…` のときソケット経由。`CALENDIT_EVENTKIT_BRIDGE_FALLBACK=1` のときのみブリッジ失敗後に `eventkit-helper` を spawn。
- **`macos doctor`**: ブリッジ経由時、JSON に `transport: "bridge"`。
