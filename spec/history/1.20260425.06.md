# 1.20260425.06

## Outlook: カレンダーグループ配下のカレンダーを `cal list` に含める

- **課題**: `GET /me/calendars` だけでは、カレンダーグループ内にのみ存在するユーザー作成カレンダーが欠ける場合がある。
- **対応**: `GET /me/calendarGroups` と各 `GET /me/calendarGroups/{id}/calendars` を `@odata.nextLink` まで取得し、`/me/calendars` の結果と **カレンダー ID で重複排除**（先に `/me/calendars` を優先）して `listCalendars` に反映。
- **実装**: [`src/services/outlook.ts`](../../src/services/outlook.ts)、共通マッパー／マージ [`src/core/outlookCalendarList.ts`](../../src/core/outlookCalendarList.ts)、ユニット [`tests/unit/outlook-calendar-merge.test.mjs`](../../tests/unit/outlook-calendar-merge.test.mjs)（`npm test` 先頭で実行）。
