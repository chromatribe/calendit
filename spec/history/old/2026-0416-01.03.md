# Version 2026-0416-01.03

## 変更内容
- JSON 中間形式の標準化（CalendarJSON フォーマット）を実装。

## 修正詳細
- `src/core/formatter.ts` に `calendarJsonSchema`（zod）、`CalendarJsonFormat` 型、`Formatter.toJson()`、`Formatter.fromJson()` を追加。
  - `toJson()`: イベント配列とメタデータ（context/service/calendar_id/generated_at/version）を含むラッパー JSON を生成。
  - `fromJson()`: ラッパー形式を zod で検証しイベントを抽出。旧来の plain array 形式も警告付きで受け入れる（後方互換）。
- `src/commands/query.ts`: `--format json` 出力を `JSON.stringify` から `Formatter.toJson()` に切り替え。コンテキスト・サービス種別のメタデータを自動付与。
- `src/commands/apply.ts`: JSON 入力の `JSON.parse` + `Partial<any>[]` を `Formatter.fromJson()` に切り替え。型安全規約に準拠。
- `spec/spec.md`: §3.2 として CalendarJSON フォーマット仕様を追記。バージョン番号を更新。
