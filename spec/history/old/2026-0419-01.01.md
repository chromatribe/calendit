# 変更履歴 `2026-0419-01.01`

## macOS Calendar（EventKit）

- **`native/eventkit-helper`**: SwiftPM 実行ファイル `eventkit-helper`（`doctor` / `list-calendars` / イベント CRUD 用サブコマンド）。CLI は `CALENDIT_EVENTKIT_HELPER` またはリポジトリ相対パスでバイナリを解決する。
- **`service: macos`**: コンテキストで EventKit の `calendarIdentifier` をカレンダー ID として指定可能。
- **`calendit macos`**: 診断とカレンダー一覧（`--json`）。

## アカウント・接続状態

- **`calendit accounts status`**: 全コンテキストを横断し、表の **`CONNECTION`** 列で接続可否を表示（Google/Outlook はトークン相当、macOS はヘルパー・TCC・カレンダー ID）。
- **`calendit auth status`**: 上記と同一の表を出力。`accounts status` 利用を推奨する案内をログに出す。

## その他

- **`add --attendees`**: カンマ区切りで参加者メールを渡せる（プロバイダにより反映可否は異なる）。
- **`docs/tests.md`**: `TC-ACCOUNTS-01`、`TC-INST-ACC-01`、無効サービス `TC-CTX-02` の期待文言（`macos` 含む）を更新。
