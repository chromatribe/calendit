# 2026-0424-01.05

## EventKit: 摩擦削減（ブリッジ自動・CLI 導線）

- **`src/core/eventkitHelper.ts`**: `CALENDIT_EVENTKIT_BRIDGE` 未指定時、macOS で `bridge.token` ＋有効な Unix ソケットがあればブリッジを自動採用。`0` / `false` 等でヘルパー固定。ブリッジ接続失敗は短いリトライ。ユーザー向け文言は i18n（`eventkit.*`）。
- **`src/core/eventkitEnvFromConfig.ts`**: `config.json` の `eventkit.defaultTransport` を、シェルに `CALENDIT_EVENTKIT_BRIDGE` が未設定のとき `process.env` へ反映（`preAction` 連動）。
- **`src/core/config.ts` / `src/types`**: `eventkit.defaultTransport`（`auto` | `bridge` | `helper`）、`set-macos-transport` 用 API。
- **`config set-macos-transport`**: 永続の既定 EventKit トランスポート。
- **`src/core/accountStatus.ts`**: `hasEventkitTransport` で「ヘルパー-only」前提を撤廃。`doctor` が `transport: "bridge"` なら `OK (bridge)`。
- **`src/commands/macos.ts`**: `macos bridge start`（.app 起動）、`macos setup`（対話でコンテキスト登録）。`doctor` / `list-calendars` は `hasEventkitTransport` 前提。`CommandDeps` を受け取る。
- **`src/core/macosBridgeApp.ts`**: `CalenditEventKitBridge.app` の候補パス解決。
- **ロケール**: en/ja キー拡充、`gen:locales` 更新。
- **ドキュメント**: `docs/commands.md`、`docs/eventkit-bridge.md`、`docs/getting-started.md`（EventKit 第9章の案内の更新予定 / 併せて本リリースで反映）。
