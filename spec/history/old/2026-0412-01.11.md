# バージョン 2026-0412-01.11

## 概要
`add` コマンドの利便性向上と安全性の強化。相対的な日付指定のサポートと Dry Run モードの追加。

## 変更・追加内容
### コマンド機能の強化
- **相対日付パースの統一**: `parseDateTime` ヘルパーを導入し、`query` と `add` 両方で `today`, `tomorrow`, `today 11:00`, `tomorrow 11:00` といった自然な指定をサポート。
- **`add` コマンドの `dry-run` 対応**: 実際に予定を入れる前に、パースされた開始・終了時刻を確認できる `--dry-run` オプションを追加。
- **ヒントメッセージの追加**: `location` や `summary` に "Zoom" 等が含まれる場合、Web会議リンクの設定が必要であることを通知するメッセージを表示。

### ロードマップの更新
- **Web Conferencing 連携**: 未来のアイデアとして `docs/idea.md` に追加。

## テスト・検証内容
- `calendit add --summary "Test" --start "tomorrow 11:00" --dry-run` で明日の11時が正しく計算されることを確認。
- `calendit query --start tomorrow` にて、追加された予定が正しく照会できることを確認。
